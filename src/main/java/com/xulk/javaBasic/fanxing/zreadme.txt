

泛型好处：
        解决存储元素的安全性问题


        定义泛型的时候，public interface List<E> extends Collection<E>
                       public class MyClass<E>{}
        在定义的时候，可以直接指定泛型E


        可以声明一个泛型方法和泛型类


        泛型的使用：
                    在集合中使用泛型
                    自定义泛型类、泛型接口、泛型方法
                    泛型与继承的关系
                    通配符

        不可以在静态方法中使用静态类型（泛型实在实例化的时候才会加载、 而静态方法通过Class可以直接调用，是class加载的时候就已经加载）
